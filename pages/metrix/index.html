행열 계산기

Description

두 테이블의 행과 열의 값을 계산합니다.

각 빈칸에 숫자를 입력하고 아래 "테이블 생성" 버튼을 누르세요!

coution

  1. 입력값에 문자와 빈값은 허용되지 않습니다.
  2. 입력값은 1 ~ 5까지의 숫자만 입력할 수 있습니다.
  3. 값을 계산하기 위해서는 두 테이블의 행과 열의 개수가 동일해야 합니다.
  4. 생성된 테이블 안에 숫자가 존재해야 계산이 가능합니다.
  5. 1. ~ 4.에 충족하지 못하면 오류 메시지가 출력 됩니다.
